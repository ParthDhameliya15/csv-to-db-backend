## Installation

1. Clone the repository:

    ```sh
    git clone https://github.com/ParthDhameliya15/csv-to-db-backend.git
    ```

2. Navigate to the project directory:

    ```sh
    cd <project_directory>
    ```

3. Install dependencies:

    ```sh
    npm install
    ```
